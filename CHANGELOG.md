## 0.0.1

- Initial release
- Port of [chenglou/pretext](https://github.com/chenglou/pretext) to Flutter/Dart
- Core APIs: `prepare()`, `layout()`, `layoutWithLines()`, `walkLineRanges()`, `layoutNextLine()`
- TextPainter-based segment measurement with caching
- CJK line breaking with kinsoku rules
- Soft hyphen, URL merging, numeric run merging
- `pre-wrap` mode support
- Example app with Editorial Engine and Bubbles demos
